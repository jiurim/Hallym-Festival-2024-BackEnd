## 📝 프로젝트 소개



 한림대힉교 학생들의 축제 참여를 더욱 편리하게 돕고 안전한 축제를 운영하기 위해 개발된 웹사이트입니다.  
  또한, **주점 예약 시스템**을 한림대 최초로 도입하여 안전성을 강화하고,<br> 다양한 정보를 효율적으로 제공하는 플랫폼을 구축했습니다.
